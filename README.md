Project Description: 
This is a mock implementation of the Path Advisor map visualization tool. 
This tool uses a screenshot of a section of the University of Utah map, 
and assigns each building as a node, and waypoints along the paths as edges to create a graph that represents the section of campus.
Using Dijkstra's Algorithm, the user can find the shortest path between two buildings on the graph/map.
The application uses streamlit, plotly, and pillow. 

To run the application:

pip install streamlit plotly pillow

streamlit run main.py
